O mně

Ahoj! Jmenuji se Michal Gulyas a jsem webový vývojář s vášní pro tvorbu moderních, funkčních a uživatelsky přívětivých webových aplikací. Od mládí mě fascinovaly technologie a jejich schopnost propojit lidi a usnadnit jim život. Tato fascinace mě přivedla na cestu vývoje webových stránek, která je dnes nejen mou profesí, ale i koníčkem.

Moje profesní zkušenosti

V průběhu mé kariéry jsem měl možnost pracovat na různých typech projektů, od malých webových prezentací až po komplexní aplikace pro firemní klienty. Specializuji se na front-end vývoj s důrazem na čistý a efektivní kód. Nejvíce se zaměřuji na HTML5, CSS3 (včetně CSS Grid a Flexboxu) a JavaScript (včetně frameworků jako React nebo Vue.js).

Dovednosti

  • HTML, CSS, JavaScript
  • Responsive Web Design
  • Práce s API a integrace třetích stran
  • Optimalizace výkonu a SEO
  • Základy backendových technologií (Node.js, Express)

Mimo profesi

Kromě programování se zajímám o fotografování a editaci fotografií. Baví mě zachycovat krásu každodenního života a objevovat nová místa. Jsem také velkým fanouškem deskových her a rád se ponořím do složitých strategických scénářů s přáteli.

Mým cílem je neustále se zdokonalovat a hledat nové výzvy, které mi umožní růst jak profesně, tak osobně. Pokud vás můj příběh zaujal, neváhejte mě kontaktovat prostřednictvím kontaktní stránky nebo se podívejte na mé portfolio.
